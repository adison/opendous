Introduction

QucsStudio is mainly a circuit simulator that has evolved out of the project Qucs, but isn't compatible with it. It's meant be be a test project to create a complete development environment for electrical engineers. The application uses the free port Q.../Free of the graphical user interface Qt® by Trolltech®.

Bug reports, comments and other help are very welcome!

Features

Currently the following applications are supported:
  • DC analysis
  • AC analysis (including noise analysis and noise distribution analysis)
  • s-parameter analysis (including noise parameter calculation)
  • transient analysis
  • Harmonic Balance analysis (including noise analysis)
  • system simulations
  • parameter sweeps and optimization of analogue circuits
  • digital simulations and Verilog simulations using ICARUS by Stephen Williams
  • PCB layout and Gerber viewer using KiCAD by the KiCAD team
  • numerical data processing using Octave by John W. Eaton et al.
  • RF transmission line calculation (coaxial, microstrip, coplanar lines etc.)
  • filter synthesis (LC ladder, microstrip, active filters etc.)
  • attenuator synthesis (resistive pi and tee topology)
  • GPIB device control
  • technical documentation of all models and types of simulation
  • support of German and English language
The following applications are planned to be implemented and tested:
  • upgrade to Qt4®
  • oscillations by Harmonic Balance analysis
  • mixed-mode analysis
  • EM field simulations
  • transistor models (HICUM, BSIM etc.)
  • VerilogAMS interface
  • many smaller improvements and extensions

Download

Currently QucsStudio supports Windows® only. It runs without installation. Decompressing the ZIP file creates the directory QucsStudio. It is run by executing QucsStudio/bin/qucs.exe.

download QucsStudio-1.0.0.zip (62 MByte)
